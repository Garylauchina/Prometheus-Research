# 进化探测式复杂系统测量框架（白皮书版）

这份文档是 Prometheus-Research 使用的“测量方法论”的**白皮书版叙事入口**。

它**不是**预测系统、控制系统或策略生成器；它是一个实验框架，用于回答一个问题：

> 在可审计合同约束下，目标复杂系统是否存在**可测量、可复述的结构**？

失败（**NOT_MEASURABLE**）是合法且重要的测量结论。

---

## 1）问题背景

现实世界复杂系统（金融市场、对抗型生态系统等）常见特征：

- 个体数量巨大、异质性强
- 局部行为高度非线性
- 宏观结构并非稳定存在，而是阶段性显现
- 传统预测/最优化/策略搜索在长期内系统性失效

---

## 2）工作假设（共同裁决结构，存活唯一）

假设 **H**：

如果所有个体共享同一类**裁决结构**（birth–death / 资源约束），但允许形成不同的内在决策坐标系，则系统宏观状态通常只呈现两种情形：

- 纯随机噪声（无可复述结构），或
- 由有限个主导有效维度描述的动力学结构（结构可显影）

该假设不要求：个体最优/理性/策略相同；只要求：**裁决统一，存活唯一**。

---

## 3）Gate 0：可测量性门（硬门槛）

任何 run 进入分析前必须先过 Gate 0：

- **0.1 干扰可归因**
  - 外部干预、规则变更、观测延迟/通道变更必须显式记录并计入裁决
  - 无法归因的强干扰 ⇒ **NOT_MEASURABLE**
- **0.2 非预设涌现**
  - 结果不得直接映射设计假设
  - 应能事后解释，但不是事前指定

Gate 0 失败 ≠ 实验失败，而是测量结论。

---

## 4）六条核心原则（极简）

- **原则1：M = N（维度对齐）**：基因维度与可观测特征按索引对应，避免隐式映射。
- **原则2：允许冗余**：宁可冗余不可遗漏，冗余由演化淘汰，**禁止人为删减**。
- **原则3：承认偏差维度**：偏差是被建模对象，不是应被抹去的噪声。
- **原则4：自然选择优先**：世界裁决个体，设计者不得指定成功路径。
- **原则5：系统出证据，观察者裁决**：系统只输出审计材料，不输出结论。
- **原则6：不存在终极解**：结论均为阶段性，持续探测是默认状态。

---

## 5）生命意义投影（关键概念）

定义：  
**生命意义投影** = 个体在统一裁决结构下，为维持自身存在而形成的内在坐标系。

关键约束：

- 不参与 loss 计算
- 不可被直接训练或优化
- 只能通过存活/淘汰被间接筛选

它不是 reward function，而是“被裁决的解释结构”。
所有决策都是生命意义的投影，不存在非理性。

---

## 6）工程实现要点（如何实现而不说谎）

- 个体最小机制：生存能量/资源是唯一硬约束
- 盈利是延续存在的手段，而非目标
- 自由度管理（Freedom budget）：
  - 低证据成本自由度优先开放
  - 高结构影响自由度延迟或冻结
  - 所有自由度必须可归因
- 策略角色：策略是探针而非答案；策略失效是结构变化信号

---

## 7）可输出与不承诺输出

该框架可稳定输出：

- 不同环境下的有效维度数变化
- 某类策略族的系统性不可存活性
- 市场/系统结构的阶段性稳定区间
- 不可测量区间的明确边界

不承诺输出：

- 稳定 alpha
- 长期最优策略
- 单次 run 的确定性结论
