# Playbook 目录（场景化一页指南）

每个场景控制在一页内：输入、步骤、关键参数、易错点、验证方式。

- **历史回测（含进化）**  
  - 数据准备、时间跨度、进化间隔、数值安全网（收益剪裁/资金上限）、对账与日志
- **OKX 虚拟盘 / 实盘**  
  - sandbox 配置、清仓流程、下单参数（tdMode/posSide/instId）、双账簿对账、频控与风控
- **创世与人口管理**  
  - family_id 分配、创世数量、移民注入、弱势家族保护、多样性阈值
- **进化调优**  
  - 繁殖/变异率、选择策略、家族多样性保护、熵指标监控
- **监督循环与执行链路**  
  - Supervisor/Moirai 调用顺序，BulletinBoard 输入输出，Ledger 写入点，DiversityMonitor 触发点
- **数值稳定性与风控**  
  - 杠杆/仓位上限、收益剪裁、资金上限、异常行情保护

后续可按需在 `docs/playbooks/` 下为每个场景补独立一页。
