# Cursor IDE 中文设置指南

本指南将帮助您将 Cursor IDE 的界面、窗口和 AI 对话全部设置为中文。

## 📋 设置步骤

### 1. 设置界面语言（UI Language）

Cursor 的界面语言需要在应用级别设置，有以下几种方法：

#### 方法一：通过命令面板（推荐）

1. 按 `Cmd+Shift+P` (macOS) 或 `Ctrl+Shift+P` (Windows/Linux) 打开命令面板
2. 输入 `Configure Display Language` 或 `配置显示语言`
3. 选择 `中文(简体)` 或 `zh-cn`
4. 重启 Cursor 使设置生效

#### 方法二：通过设置界面

1. 点击左下角齿轮图标 ⚙️ 或按 `Cmd+,` (macOS) / `Ctrl+,` (Windows/Linux)
2. 在搜索框中输入 `locale` 或 `language`
3. 找到 `Locale` 设置项
4. 选择 `zh-cn` (中文简体)
5. 重启 Cursor

#### 方法三：直接编辑配置文件

1. 按 `Cmd+Shift+P` / `Ctrl+Shift+P` 打开命令面板
2. 输入 `Preferences: Open User Settings (JSON)`
3. 添加或修改以下配置：
```json
{
  "locale": "zh-cn"
}
```
4. 保存并重启 Cursor

### 2. 安装中文语言包（如果需要）

如果上述方法无法完全中文化，可能需要安装中文语言包：

1. 打开扩展市场（Extensions）
   - 点击左侧边栏的扩展图标
   - 或按 `Cmd+Shift+X` (macOS) / `Ctrl+Shift+X` (Windows/Linux)
2. 搜索 `Chinese (Simplified) Language Pack`
3. 点击 `Install` 安装
4. 重启 Cursor

### 3. AI 对话语言设置

AI 对话的语言已经在项目配置中设置为中文。如果您想确保 AI 始终使用中文回复：

1. 在 Cursor 设置中搜索 `AI` 或 `Assistant`
2. 找到 AI 相关的语言设置
3. 或者在每次对话开始时明确要求使用中文

**注意**：根据项目配置，AI 助手已经设置为默认使用中文简体回复。

### 4. 验证设置

重启 Cursor 后，您应该看到：
- ✅ 菜单栏和界面文字变为中文
- ✅ 右键菜单为中文
- ✅ 设置界面为中文
- ✅ 错误提示和通知为中文

## 🔧 工作区配置

项目已包含 `.vscode/settings.json` 配置文件，其中包含：
- UTF-8 文件编码设置
- 中文区域设置提示
- 终端配置

这些设置会在打开此工作区时自动应用。

## ❓ 常见问题

### Q: 设置后界面仍然是英文？
A: 
1. 确保已完全重启 Cursor（不是重新加载窗口）
2. 检查是否安装了中文语言包
3. 尝试清除 Cursor 缓存后重新设置

### Q: 部分界面仍然是英文？
A: 某些插件或功能可能没有完全本地化，这是正常现象。核心界面应该已经中文化。

### Q: 如何恢复为英文？
A: 按照相同步骤，将语言设置改回 `en` 或 `en-us` 即可。

## 📝 额外提示

- 如果某些扩展或功能没有中文版本，这是正常的
- 代码编辑器的核心功能（如语法高亮、代码补全）不受语言设置影响
- 终端输出和代码内容本身不会因界面语言而改变
