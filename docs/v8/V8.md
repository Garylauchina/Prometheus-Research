## 元判断：可测量性判据

### 原则0

**在应用V8.0方法之前，首先判断目标系统是否可被测量：**

**两重判据：**

**0.1 干扰可测量**
- 观测系统对目标系统的干扰是可量化或可比较的

**0.2 涌现规律**
- 观测结果存在自发涌现的规律
- "涌现" = 非人为预设的、意外的、事后可解释的
- 如果只是重复预设，那是验证，不是测量

**如果满足以上判据 → 系统可被测量 → 继续应用原则1-6**

**哲学基础：操作主义**
> 一个概念的意义 = 测量它的操作
> 
> 不问"理论上能否测量"，只问"实际上测到了什么"
> 
> 让实践说话

## Meta-Judgment: Measurability Criteria

### Principle 0

**Before applying V8.0, first determine if the target system is measurable:**

**Dual Criteria:**

**0.1 Disturbance is Measurable**
- The disturbance of the observation system on the target system is quantifiable or comparable

**0.2 Emergent Patterns**
- Observation results exhibit spontaneously emergent patterns
- "Emergent" = non-preset, unexpected, explainable post-hoc
- If only repeating presets, that's validation, not measurement

**If these criteria are met → system is measurable → proceed to Principles 1-6**

**Philosophical Foundation: Operationalism**
> The meaning of a concept = the operation to measure it
> 
> Don't ask "theoretically measurable", ask "what was actually measured"
> 
> Let practice speak

## Core Principles

**Principle 1: M = N (Gene dimension aligns with world features)**

Gene[i] ↔ Feature[i], position-aligned, weights evolve

**Principle 2: Allow redundant dimensions**

N >> true effective dimensions, better redundant than missing, evolution auto-reduces

**Principle 3: World features = Measurable dimensions + Measurement bias dimensions**

Acknowledge measurement imperfection, provide correction space, evolution reveals truth

**Principle 4: Measurement system evolution follows natural selection**

Objective world is the only evaluation standard, no human intervention

**Engineering formulation (aligned with V10)**:
- If selection pressure is dominated by **ecological rules** (not strategy fences / designer gates), population behavior should converge into a small set of **dominant modes** in some phases;
- These modes should be **repeatably observable** across windows/seeds (allowing regime switching).

**Principle 5: The Measurement System Demonstrates, the Observer Determines**

The measurement system evolves and presents data; the observer judges the system's nature (static/dynamic/chaotic)

**Principle 6: Better solutions always exist**

Current answer is not ultimate truth, continuous exploration
