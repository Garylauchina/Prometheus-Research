## 🐛 异常监控

### **需要警惕的问题**

❌ **问题1：过度保守**
```
症状：大部分周期无交易，观望率>80%
原因：开仓门槛过高
解决：降低min_confidence_to_open（65% → 60%）
```

❌ **问题2：频繁止损**
```
症状：每个Agent平均持仓<5周期就止损
原因：止损阈值过低
解决：放宽普通反向趋势止损（-0.8% → -1.0%）
```

❌ **问题3：仍然全面亏损**
```
症状：所有Agent PnL仍为负
原因：预言仍然滞后，或市场环境极端
解决：
1. 检查预言的recent_price_momentum权重
2. 考虑引入"反向操作"模式（预言反转）
```
